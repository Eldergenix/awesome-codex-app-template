# Notion and Linear Integration

## Notion

Use Notion MCP for documentation, packages, notes, and progress. Keep the database schema aligned with `docs/notion/schemas/`.

## Linear

Use Linear MCP for task creation, task updates, bug tracking, and completion comments. Every task should have acceptance criteria and verification commands.

## Fallback

If MCP is unavailable, create drafts locally:

- Linear: `docs/linear/issue-template.md`
- Notion: `docs/notion/schemas/` and `docs/progress/`
