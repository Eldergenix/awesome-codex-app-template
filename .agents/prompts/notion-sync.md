# Notion Sync Prompt

Use Notion MCP to update the workspace. Maintain these surfaces:

1. Documentation: architecture, PRD, API, runbook, ADR.
2. Packages: package ownership, dependency risk, changelog.
3. Notes: decisions, investigations, retrospectives.
4. Progress: task status, verification, blockers, next action.

If MCP is unavailable, create a local draft in `docs/` and mention that it still needs sync.
