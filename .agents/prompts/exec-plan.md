# ExecPlan Prompt

You are implementing a non-trivial change. Produce or update an ExecPlan in `PLAN.MD` before coding.

Required output:

- objective
- linked Linear issue or draft payload
- linked Notion doc or draft target
- scope and non-goals
- acceptance criteria
- implementation steps
- verification commands
- rollback plan
- risk register

Do not implement until the plan has concrete verification steps.
