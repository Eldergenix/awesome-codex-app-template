## Instructions

Use Linear as the task source of truth.

Required issue fields:

- Title
- Type: bug, feature, task, chore, security
- Acceptance criteria
- Verification commands
- Risk level
- Links to Notion documentation
- Definition-of-done checklist

Completion requires updating the issue with changed files, commands run, unresolved risks, and final status.
