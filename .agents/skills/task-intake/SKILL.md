1. Read the root prompt files and nearest nested `AGENTS.md`.
2. Convert the request into objective, scope, non-goals, acceptance criteria, and verification commands.

4. Create or update the Linear issue through MCP when connected. If MCP is unavailable, produce a draft Linear payload.
5. Create or update the Notion progress entry through MCP when connected. If MCP is unavailable, update `docs/progress/`.
